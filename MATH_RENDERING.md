# Math Rendering in Chat

Your chat now supports beautiful mathematical notation using KaTeX!

## How to Use Math in Chat

### Inline Math
Use single dollar signs `$...$` for inline math:
```
The quadratic formula is $x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$
```

### Display Math (Block)
Use double dollar signs `$$...$$` for centered, larger math:
```
$$
\int_{a}^{b} f(x) \, dx = F(b) - F(a)
$$
```

## Common Math Examples

### Fractions
```
$\frac{5}{6}$ or $\frac{numerator}{denominator}$
```

### Square Roots
```
$\sqrt{x}$ or $\sqrt[3]{x}$ for cube root
```

### Exponents and Subscripts
```
$x^2$ or $e^{i\pi}$ or $a_n$ or $x_{i,j}$
```

### Greek Letters
```
$\alpha$, $\beta$, $\gamma$, $\Delta$, $\pi$, $\theta$
```

### Summation and Products
```
$\sum_{i=1}^{n} i^2$ or $\prod_{i=1}^{n} i$
```

### Limits
```
$\lim_{x \to \infty} f(x)$
```

### Matrices
```
$$
\begin{pmatrix}
a & b \\
c & d
\end{pmatrix}
$$
```

### Trigonometry
```
$\sin(x)$, $\cos(x)$, $\tan(x)$
```

### Calculus
```
$\frac{dy}{dx}$ or $\frac{\partial f}{\partial x}$
```

## Ask the AI
The AI can now respond with properly formatted mathematical equations. Just ask math questions naturally, and the AI will format its responses with proper mathematical notation!

Example:
```
User: "What's the Pythagorean theorem?"
AI: "The Pythagorean theorem states that in a right triangle: $$a^2 + b^2 = c^2$$"
```
